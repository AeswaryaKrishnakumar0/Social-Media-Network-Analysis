# Social-Media-Network-Analysis

This project performs a social network analysis of Mastodon users who posted using the hashtag #WSUCOMP7025. Using the R programming language and the Mastodon API, posts containing the hashtag were collected and analysed to identify the servers generating the most activity. A directed social network graph was constructed where each node represents a user and edges represent follower relationships. Network analysis techniques were then applied to understand the structure of the network, including identifying connected components and visualising the largest component of the graph. 

Further analysis was conducted to evaluate key network properties such as graph diameter, density, PageRank popularity scores, and homophily based on lunch location data. The results show that the network exhibits characteristics of a scale-free structure, where a small number of users act as influential hubs while most users have relatively few connections. The PageRank analysis helped identify the most influential accounts within the network, while the homophily analysis indicated that users do not strongly prefer to connect with others who share the same lunch location.
